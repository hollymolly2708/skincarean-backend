## CART API SPEC

## CREATE CART

Endpoint : POST /api/cartItem

Request Header :

- X-API-TOKEN : Token

Request Body :

```json
{
  "user_id": "123232",
  "product_id" : "232323"
}
```

Response Body (Success) :

```json
{
  "data": "Product berhasil ditambahkan ke keranjang"
}
```

Response Body (Failed) :

```json
{
  "data": "Unauthorized"
}
```

## GET ALL CART

Endpoint : GET /api/cartItem

- X-API-TOKEN : Token

Response Body (Success) :

```json
{
  "data": [
    {
      "id": "232323",
      "user_id": "123232",
      "product_id": "232323"
    }
  ]
}
```

Response Body (Failed) :

```json
{
  "errors": "unauthorized"
}
```
